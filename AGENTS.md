# SwiftBar Menu Bar Scripts

This directory contains menu bar scripts for [SwiftBar](https://swiftbar.app/), a macOS menu bar plugin host.

Scripts in this folder appear as items in the macOS menu bar. SwiftBar supports scripts in any executable language (bash, python, ruby, etc.) and uses filename conventions to control refresh intervals (e.g., `script.5m.sh` refreshes every 5 minutes).
